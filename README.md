# .zsh

my zsh plugin directory

## How to use

```sh
cd ~
git clone https://github.com/wmartinmimi/.zsh.git
cd .zsh
git submodule init && git submodule update
fast-theme -t ~/.zsh/zsh-fsh/themes/catppuccin-mocha
echo 'source ~/.zsh/init.sh' >> .zshrc
```
